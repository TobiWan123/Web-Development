What is W3C?


The World Wide Web Consortium (W3C) is the main international standards organization for the World Wide Web. Founded in 1994 and currently led by Tim Berners-Lee, the consortium is made up of member organizations that maintain full-time staff working together in the development of standards for the World Wide Web.



This project is also completly static, which means it is really easy to host and has lightweight code without PHP for example. Static in this context means that this project does not posses a backend. Because of this I also needed to implement a special functionality to receive messages to my google account through the contact formular of this site. Normally you would use a function mail() in PHP (dynamic, backend) or other non static approaches, but this is not that easy to host and with a SMTP-Server for mailing mostly not free. Some hosting-services include the possibility to receive mails through the service but this also comes with costs in some cases. So the best opportunity for me was to do it by myself. With Google-Sheets I could fix this problem. This Website is for an international audience so it is not multilingual, but maybe I'll add some additional languages like German and Chinese in the future.
